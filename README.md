# Real-Time Chat Application

A real-time chat application built using Node.js, Express and Socket.io.

## Features
- Real-time messaging
- Online users list
- Typing indicator
- Emoji support
- User join/leave notification

## Tech Stack
- Node.js
- Express.js
- Socket.io
- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository
2. Install dependencies

npm install

3. Run the server

node server.js

4. Open browser

http://localhost:3000
